## Notes Processing Preferences

### Language and Style
- Язык: русский
- Терминология: медицинская (терапия, ревматология, АБТ)
- Формат: краткий, без воды, с конкретными цифрами и сроками

### Structure
Preferred format:
```
Задача → Срок → Время на выполнение → Результат
```

### Prioritization
- **Urgent (p1-p2)**: Anki-повторение, уроки с отставанием, новые уроки в день выхода
- **Strategic (p3-p4)**: перевод старых уроков в Anki, текстовые материалы, посты

## Anti-Patterns (AVOID)

When processing entries DO NOT:
- Планировать без учёта реального времени (1 карточка = 1 минута, 1 урок = 2–2.5 часа)
- Ставить слишком много задач на день
- Игнорировать рост времени на Anki-повторение
- Планировать зал в период нагона отставания
- Разрешать кино/сериалы/игры до выполнения дневного плана
